## What is AI Workdesk?

AI Workdesk is a professional RAG (Retrieval-Augmented Generation) platform that combines:

- **Smart Document Processing** - Ingest PDFs, web pages, YouTube videos, and more
- **Advanced RAG Techniques** - Multiple retrieval strategies for optimal results
- **Knowledge Graphs** - Visualize relationships in your data
- **Multi-Agent Workflows** - Orchestrate complex AI tasks
- **Privacy-First** - Run entirely locally with Ollama
